# 🚀 GitScan

GitScan is a GitHub profile analyzer that fetches public account data from the GitHub REST API and turns it into a clean, dashboard-style overview. It helps users quickly inspect profile details, repository activity, language usage, and overall project stats.

---

## 🌍 Live Demo

https://sevanton88.github.io/gitscan/

---

## ✨ Features

- Search any public GitHub username directly from the landing screen
- View profile details such as avatar, bio, location, company, website, and join date
- Analyze repository totals including stars, forks, watchers, and overall project size
- See a language breakdown, top repositories by stars, and recently updated repositories

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- GitHub REST API

---

## ⚡ Getting Started

1. Clone the repository  
2. Open `index.html` in your browser
